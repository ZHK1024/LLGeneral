## Installation

> 使用之前先添加私有源: [查看说明](https://github.com/ZHK1024/LLSpecs)

LLGeneral is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LLGeneral'
```

结构
```
.
├── Extensions
├── File

```
